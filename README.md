# CharSCNN
## About
This is implementation of CharSCNN and SCNN.


## Requirement
- Python 2 >= 2.6
- NumPy >= 1.7.1
- SciPy >= 0.11
- Theano == 0.9.0
- scikit-learn == 0.17
	- for dividing dataset into training set and test set


## Usage


To run SCNN;
```
python scnn.py
```

To run CharSCNNl;
```
python charscnn.py
```

Note: default dataset size (i.e., `num_sent`) is set to 20,000 in `load.py` or `char_load.py`
